# User Management Project

This is a Node.js project for managing users in a community.  
It allows CRUD operations on user data and is built using Express.js.

## Features
- Add, update, delete, and list users
- Simple REST API
- Uses `.env` for environment variables
- Ignores unnecessary files via `.gitignore`

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `node usermanagement.js` to start the server
